# Gerenciamento-de-Processos-e-Escalonamento
Compreender o funcionamento do gerenciamento de processos em um sistema operacional, incluindo criação, controle e escalonamento. O trabalho visa implementar um simulador de escalonamento de processos e explorar conceitos como troca de contexto, estados de processo, métricas de desempenho e problemas de concorrência (race condition, deadlock, etc.)
